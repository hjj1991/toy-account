# Welcome  뜨끔한 가계부 🤑

> 뜨끔한 가계부 / 개인 토이프로젝트

🏠 [사이트 바로가기](https://cash.sundry.ninja)

## 📄Description
```sh
뜨끔한 가계부는 재테크가 중요한 시대의 다양한 금융정보를 사용자에게 제공 하며, 가계부 작성을 통한 소비습관 증진에 도움을 주기위한 사이트 제작을 목표로 하고 있으며 끊임없이 업데이트 할 예정입니다.
```
